## 📋 About tauri-plugin-macos-fps

tauri-plugin-macos-fps is a tool designed for macOS users who want smoother animation and better display performance in apps built with Tauri. It removes the usual 60fps (frames per second) limit in WKWebView, which is the system responsible for displaying web content in macOS apps. This allows apps to run at refresh rates above 60fps, improving the experience especially on newer Mac screens with higher refresh rates.

While this plugin is made for developers using Tauri, this guide shows how you can get it set up and running on your Mac without needing technical knowledge.

## ⚙️ How It Works

Normally, macOS limits web views inside apps to 60fps to save power and keep the system stable. This plugin changes that limit to unlock higher frame rates. Once installed, web content inside Tauri apps can refresh as fast as your display supports, making animations smoother and reducing lag.

## ❓ Troubleshooting

- **The app still runs at 60fps:** Ensure you restarted the app after plugin installation. Some apps require a full restart of your Mac.  
- **Permission errors during installation:** Go to System Preferences and give authentication or security permissions to the installer app.  
- **Plugin does not seem to activate:** Make sure you downloaded the latest version. Try downloading again from the GitHub page.  
- **Unexpected app crashes or freezes:** The higher frame rate can increase power use. Close other apps to free system resources.
